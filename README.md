# NIT3213 Data Viewer App

## 📱 Overview
This is an Android application developed for NIT3213 assignment.  
The app allows users to login, view dashboard data, and see detailed information.

---

## 🚀 Features
- Login authentication
- Dashboard with list of items
- Details screen
- MVVM Architecture (ViewModel + LiveData)
- Dependency Injection using Koin
- Unit Testing using JUnit

---

## 🏗 Architecture
- MVVM (Model-View-ViewModel)
- Clean separation of UI and data logic

---

## 🛠 Technologies Used
- Kotlin
- Retrofit (API calls)
- Koin (Dependency Injection)
- RecyclerView
- JUnit (Unit Testing)
